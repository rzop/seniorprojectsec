To run the new AI helper component:

1. You will also need to go to this website:
 
 https://console.groq.com/keys 
 
and generate your own groq api key/ You will copy and paste that into the GROQ_API_KEY variable 
within the backend\.env file. You should save this key for future use.

2. You will need to run this in your terminal:

npm install

3. In addition to the normal "python start_backend.py" you may also need to explicilty run:

python app.py
